# Чек-лист: Форма авторизации (Login)

**URL:** https://demo.prv029.com/documented/24/index.php

## Основные элементы (по требованиям)
- [ ] Кнопка "Register" присутствует и кликабельна
- [ ] Поле "Email" присутствует
- [ ] Поле "Password" присутствует
- [ ] Кнопка "Log in" присутствует

## Функциональность авторизации
### Позитивные сценарии
- [ ] Вход с зарегистрированным email и правильным паролем
- [ ] После успешного входа отображается "Home" страница
- [ ] На Home странице есть надпись "Signed in as [имя]"
- [ ] На Home странице есть кнопка "Log out"

### Негативные сценарии
- [ ] Неверный email + правильный пароль → ошибка
- [ ] Правильный email + неверный пароль → ошибка
- [ ] Неверный email + неверный пароль → ошибка
- [ ] Пустое поле Email → ошибка
- [ ] Пустое поле Password → ошибка
- [ ] Оба поля пустые → ошибка

## Сообщения об ошибках (требование #6)
- [ ] При ошибке показывается сообщение: "Please enter your Email address and password correct"
- [ ] Поля Email и Password очищаются после ошибки
- [ ] Сообщение об ошибке понятное и заметное

## Безопасность
- [ ] Пароль отображается точками (требование #5)
- [ ] Нет подсказки, что именно неверно (email или пароль)
- [ ] Сообщение об ошибке не раскрывает информацию о существовании пользователя

## Юзабилити
- [ ] Можно переключаться между полями по Tab
- [ ] Кнопка Log in доступна по Enter
- [ ] Кнопка Register ведет на страницу регистрации

## Сессия и выход
- [ ] Кнопка Log out работает корректно
- [ ] После Log out открывается страница логина
- [ ] Сессия завершается полностью (нельзя вернуться по Back)
