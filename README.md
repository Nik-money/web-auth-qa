# Исследовательское тестирование формы регистрации и авторизации

## Цель проекта
Провести исследовательское тестирование веб-форм регистрации и авторизации, документировать процесс и найденные дефекты.

## Объект тестирования
*   URL тестового сайта: [https://demo.guru99.com/insurance/v1/index.php]
*   Форма регистрации 
*   Форма авторизации

## Методология
*   Исследовательское тестирование.
*   Составление чек-листов, тест-кейсов и баг-репортов.
*   Тестирование на соответствие требованиям, удобство использования, безопасность.

## Структура репозитория
*   `1_Test_Strategy_&_Plan/` - Подход и план тестирования.
*   `2_Checklists/` - Чек-листы.
*   `3_Test_Cases/` - Тест-кейсы.
*   `4_Bug_Reports/` - Баг-репорты.
*   `5_Test_Data/` - Наборы тестовых данных.
*   `6_Conclusions_&_Report/` - Итоговый отчет.
