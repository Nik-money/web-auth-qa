# Чек-лист: Сессия и профиль пользователя

**Контекст:** После успешного входа в систему

## Home страница (требование #4)
- [ ] Отображается надпись "Signed in as [имя пользователя]"
- [ ] Кнопка "Log out" присутствует и доступна
- [ ] Навигационное меню присутствует:
  - [ ] Home
  - [ ] Request Quotation
  - [ ] Retrieve Quotation
  - [ ] Profile
  - [ ] Edit Profile
- [ ] Футер с информацией: "Broker Insurance WebPage"
- [ ] Дата последнего изменения: "Last modified: November 01 2019 11:00:04"

## Управление сессией
### Выход из системы
- [ ] Кнопка "Log out" завершает сессию
- [ ] После выхода показывается страница логина
- [ ] Нельзя вернуться к защищенным страницам по кнопке Back
- [ ] При попытке доступа к Home после выхода требуется авторизация

### Timeout сессии
- [ ] Через некоторое время бездействия сессия завершается
- [ ] При истечении сессии показывается сообщение/редирект на логин
- [ ] Данные формы не теряются при коротком timeout (если есть)

## Профиль пользователя
### Страница Profile
- [ ] Доступна из навигационного меню
- [ ] Отображает информацию пользователя (из формы регистрации)
- [ ] Данные отображаются корректно (совпадают с введенными при регистрации)

### Страница Edit Profile
- [ ] Доступна из навигационного меню
- [ ] Поля предзаполнены текущими данными
- [ ] Можно редактировать данные профиля
- [ ] Сохранение изменений работает корректно
- [ ] После сохранения данные обновляются на Profile странице

## Навигация
- [ ] Все пункты меню ведут на соответствующие страницы
- [ ] Текущая страница подсвечена в меню
- [ ] На каждой странице есть возможность выйти (Log out)
- [ ] Консистентность меню на всех страницах

## Безопасность
- [ ] Нет доступа к URL защищенных страниц без авторизации
- [ ] Cookie/сессионные данные защищены
- [ ] Пароль не отображается в открытом виде в профиле
