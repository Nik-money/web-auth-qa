# Чек-лист: Форма регистрации (Register)

**Переход:** Нажать "Register" на странице логина

## Структура формы
### Личная информация
- [ ] Поле "Title" (выпадающий список: Mr/Mrs/Miss и т.д.)
- [ ] Поле "First name" (текстовое)
- [ ] Поле "Surname" (текстовое)
- [ ] Поле "Phone" (текстовое)
- [ ] Поле "Date of Birth" (группа выпадающих списков: год, месяц, день)
- [ ] Радио-кнопки: "Full" и "Provisional"

### Информация о лицензии
- [ ] Поле "Licence Period" (выпадающий числовой список: от 1 до 80 + текст "years")
- [ ] Поле "Occupation" (выпадающий список: Academic, Actor, Artist и другие значения)
- [ ] Поле "Address" (Street, City, County, Post code)
- [ ] Поле "Post code": (отображает плейсхолдер: "sy24 5be")

### Учетные данные
- [ ] Поле "Email" (отображает плейсхолдер: "youremail@abv.bg)
- [ ] Поле "Password"
- [ ] Поле "Confirm password"
- [ ] Кнопки: "Reset" и "Create"

## Валидация полей
### Обязательные поля
- [ ] Все поля обязательны (или есть необязательные?)
- [ ] Сообщение об ошибке при пустых обязательных полях

### Специфичные проверки
- [ ] Email: проверка формата (должен содержать @)
- [ ] Email: уникальность (нельзя зарегистрировать существующий)
- [ ] Password: минимальная длина
- [ ] Password: соответствие требованиям безопасности
- [ ] Confirm password: должно совпадать с Password
- [ ] Phone: допустимый формат номера
- [ ] Date of Birth: валидная дата (не будущая, не слишком ранняя)
- [ ] Post code: валидный формат

## Кнопки
- [ ] Кнопка "Create" отправляет форму при валидных данных
- [ ] Кнопка "Reset" очищает все поля формы
- [ ] После успешной регистрации происходит автоматический вход
- [ ] После успешной регистрации показывается Home страница

## Юзабилити
- [ ] Логичный порядок заполнения полей (сверху вниз)
- [ ] Auto-focus на первом поле при загрузке
- [ ] Можно переключаться между полями по Tab
- [ ] Подсказки/плейсхолдеры в полях
