# Тестовые данные

В этой папке хранятся наборы тестовых данных для тестирования форм регистрации и авторизации.

## Структура:
- `Credentials.md` - Учетные данные для тестирования
- `Test_Data_Sets.md` - Наборы данных для позитивного/негативного тестирования
- `Boundary_Values.md` - Граничные значения для полей
- `Registered_Users.md` - Информация о зарегистрированных тестовых пользователях
- `Actual_Test_Data.md` - история реального тестирования
## Использование:
1. Для выполнения тест-кейсов используйте данные из соответствующих файлов
