# Граничные значения для полей формы

## Поле First name / Surname
- **Минимальная длина:** 1 символ
- **Максимальная длина:** [нужно определить тестированием]
- **Граничные значения:**
  - Пустая строка: "" (не должно приниматься)
  - 1 символ: "A"
  - 2 символа: "Ab"
  - 50 символов: "A".repeat(50)
  - 255 символов: "A".repeat(255) (проверить ограничение)

## Поле Email
- **Минимальная длина:** 6 символов (a@b.cd)
- **Максимальная длина:** 254 символа (стандарт RFC)
- **Граничные значения:**
  - Минимальный валидный: "a@b.cd"
  - Максимальный локальная часть: 64 символа
  - Максимальный домен: 255 символов
  - Спецсимволы в локальной части: "user+tag@domain.com"

## Поле Password
- **Минимальная длина:** [определить тестированием, предположительно 6]
- **Максимальная длина:** [определить тестированием]
- **Требования (предположительные):**
  - Минимум 1 заглавная буква
  - Минимум 1 строчная буква
  - Минимум 1 цифра
  - Минимум 1 специальный символ
  - Без пробелов

## Поле Phone
- **Формат:** международный
- **Минимальная длина:** 10 цифр (без кода страны)
- **Максимальная длина:** 15 цифр (с кодом страны)
- **Допустимые символы:** цифры, +, -, пробелы, скобки

## Поле Date of Birth
- **Минимальная дата:** 1900-01-01 (предположительно)
- **Максимальная дата:** сегодня - 18 лет (для совершеннолетия)
- **Форматы:**
  - Выбор из календаря

## Поле Post code
- **Формат UK:** [A-Z]{1,2}[0-9][A-Z0-9]? [0-9][A-Z]{2}
- **Пример:** "SW1A 1AA", "M1 1AA", "B33 8TH"
- **Значение по умолчанию:** "sy24 5be" (из UI)
