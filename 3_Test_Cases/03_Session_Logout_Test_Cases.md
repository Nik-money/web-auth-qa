**Приоритет:** High  
**Предусловие:** Пользователь авторизован в системе (email: qa_test_nikita@mail.ru)  
**Шаги:**
1. Убедиться, что отображается страница "Home"

## TC-LOGOUT-003: Проверка навигационного меню после входа
**ID:** TC-LOGOUT-003  
**Приоритет:** Medium  
**Предусловие:** Пользователь авторизован в системе  
**Шаги:**
1. После успешного входа (TC-LOGIN-001) проверить наличие навигационного меню
2. Проверить все пункты меню: Home, Request Quotation, Retrieve Quotation, Profile, Edit Profile

- Все пункты меню отображаются корректно
- Каждый пункт ведет на соответствующую страницу
