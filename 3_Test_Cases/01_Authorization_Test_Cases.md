# Тест-кейсы: Форма авторизации (Login)

## TC-LOGIN-001: Успешная авторизация после регистрации
**ID:** TC-LOGIN-001  
**Приоритет:** Critical  
**Предусловие:** Пользователь зарегистрирован в системе (email: qa_test_nikita@mail.ru, пароль: TestPass123!)  
**Шаги:**
1. Перейти на страницу авторизации ([https://demo.prv029.com/documented/24/index.php](https://demo.guru99.com/insurance/v1/index.php))
2. В поле "Email" ввести: qa_test_nikita@mail.ru
3. В поле "Password" ввести: TestPass123!
4. Нажать кнопку "Log in"
   
**Ожидаемый результат:** 
- Пользователь перенаправлен на страницу "Home"
- Отображается надпись "Signed in as [имя пользователя]" (ожидаемо Nikita или подобное)
- В меню доступна кнопка "Log out"
  
**Фактический результат:** Пользователь перенаправлен на страницу "Home"

**Статус:** Pass
