# Баг-репорты

В этой папке хранятся реальные баг-репорты, найденные в ходе исследовательского тестирования форм регистрации и авторизации.

## Найденные дефекты:

### BR-001: Критическая уязвимость безопасности
**Приоритет:** Critical  
**Проблема:** Возможность входа в систему с пустыми полями Email и Password  
**Влияние:** Любой пользователь может получить доступ к системе без аутентификации  
**Статус:** New  
**Файлы:** `BR-001_screenshot1.png`, `BR-001_screenshot2.png`

### BR-002: Потеря данных пользователя
**Приоритет:** High  
**Проблема:** Данные профиля не сохраняются и не отображаются после регистрации  
**Влияние:** Пользователь не видит свои данные, возможна потеря информации  
**Статус:** New  
**Файлы:** `BR-002_screenshot3.png`, `BR-002_screenshot4.png`

### BR-003: Некорректное отображение
**Приоритет:** Medium  
**Проблема:** Вместо имени пользователя отображается email на главной странице  
**Влияние:** Нарушение требований и ухудшение пользовательского опыта  
**Статус:** New  
**Файлы:** `BR-003_screenshot5.png`
