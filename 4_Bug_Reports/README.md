# Баг-репорты

В этой папке хранятся реальные баг-репорты, найденные в ходе исследовательского тестирования.

## Найденные дефекты:

### BR-001: Критическая уязвимость безопасности
**Проблема:** Возможность входа в систему с пустыми полями авторизации  
**Влияние:** Любой пользователь может получить доступ без пароля  
**Статус:** New

### BR-002: Потеря данных пользователя
**Проблема:** Данные профиля не отображаются после регистрации  
**Влияние:** Пользователь не видит свои данные, возможна потеря информации  
**Статус:** New

### BR-003: Некорректное отображение
**Проблема:** Вместо имени пользователя отображается email  
**Влияние:** Нарушение требований и ухудшение пользовательского опыта  
**Статус:** New
